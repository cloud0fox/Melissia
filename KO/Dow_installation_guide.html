▲ 계정 생성 클릭.


▲ ※ 확인대는 내용을 적고 계정을 생성하시면 됩니다.



2. 설치파일 다운로드

회원가입도 했겟다. 이제 클라이언트 설치하로 가봅시다.!

▲ 클라이언트 다운로드를 들어가시면 구글 다운로드로 진행합니다
※ 다운받을수 없다고 하시면 구글 로그인 하시고 받아보세요!

또는 더이상 다운받을수 없다고 뜨면
구글 드라이브 다운로드 용량 초과 오류 발생 시 우회방법 모음 ( 글자를 클릭 ) 을 확인해주세요!
또는 오류 해결법 맨 아래쪽에 있습니다

3. 런처파일 다운로드

이제 런처파일을 받아봅시다

▲ 서버 런처 다운로드

▲ 수동 풀 패키지 다운로드
▲ 압축을 풀고 나온 파일들을 Melissia 폴더안으로 모두 넣으시면 됩니다.
 
 

4. 한글패치 하기

이제 대망의 한글패치를 해봅시다. 추가적인 한글패치 입니다 안하셔도 무방합니다

▲ 디스코드 게시판중 한글패치 게시판을 눌러주세요.( 최근에 올라온 파일을 받아봅시다 )

▲ 다운로드를 받으신후 Melissia 설치폴더 에서 ads 폴더 안에 넣어줍니다.
이후 런처 실행하구 게임하시면 됩니다
